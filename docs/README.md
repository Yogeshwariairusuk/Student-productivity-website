#  Student Productivity Tool

A full-stack **Student Productivity Management System** that helps students
plan, track, and succeed in their academic journey.

Built with **React + FastAPI + PostgreSQL**.


## Features

### Dashboard
- Total subjects, tasks, assignments
- Completed vs pending tasks
- Attendance alerts (<75%)

### Subjects
- Add, edit, delete subjects
- Real-time updates

### Tasks
- Add tasks linked to subjects
- Track status (Pending / Completed)
- Edit & delete tasks

### Assignments
- Assignment tracking by subject
- Due dates & status

### Attendance
- Attendance percentage calculation
- Low attendance alerts

## Tech Stack

 Layer - Technology 
-------------------
Frontend - React + Vite 
Backend - FastAPI 
Database - PostgreSQL 
ORM - SQLAlchemy 
Styling - CSS
